# Linux Text Editing Exercise

## Overview
This project demonstrates basic Linux text editing using **nano** and **vi/vim** editors. It focuses on creating, editing, and managing files, as well as understanding and applying **file permissions**.

## Objective
- Learn to create new files using the `touch` command.
- Edit files with **nano** (beginner-friendly) and **vi/vim** (advanced).
- Manage file permissions with `chmod`.
- Use essential Linux commands like `ls` and `pwd`.

## Files Included
- `class_notes.txt` – Contains notes and examples of commands learned during the exercise.
- `README.md` – Project description and instructions.

## Skills Demonstrated
- Creating and editing files in Linux.
- Navigating the Linux filesystem.
- Using nano and vi/vim editors effectively.
- Understanding and modifying file permissions (`chmod`).
- Applying basic Linux commands (`ls`, `pwd`, `touch`).
