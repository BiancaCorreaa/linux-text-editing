# Linux Text Editing Exercise

This project demonstrates basic text editing in Linux using the nano and vi/vim editors. 
It includes exercises for creating, editing, and managing file permissions. 

## Files included:
- `class_notes.txt`: Contains notes and examples of commands learned.
### Skills Demonstrated
- Creating and editing files in Linux
- Using nano and vi/vim editors
- Managing file permissions with chmod
- Using basic Linux commands (ls, pwd, touch)
