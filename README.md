# Linux Text Editing Exercise

## Overview
This project demonstrates basic Linux text editing using **nano** and **vi/vim** editors, focusing on file creation, editing, and managing file permissions.

## Skills Demonstrated
- Creating/editing files with `nano` and `vi/vim`
- Managing file permissions (`chmod`)
- Using basic Linux commands: `touch`, `ls`, `pwd`

## Files
- `class_notes.txt` – Notes and examples from the exercise.
- `README.md` – Project description.
